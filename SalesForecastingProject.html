Walmart Sales Forecasting Project

Project Overview

This project is part of the M5 Forecasting Challenge, focusing on predicting unit sales for Walmart stores across three US states: California, Texas, and Wisconsin. The goal is to estimate daily sales for the next 28 days using historical sales data and advanced forecasting techniques.
